More about my programming journey

I started programing when I was 12-13 years old in visual basic using visual studio making windows form applications, after a while and dozen of small applications I started learning C#. I was also usign C# to make windows for applications and even made some games.

Web development

Next step in my programming journey were learning HTML, CSS, JS, React and Angular. I've spent more than 2 years doing only frontend until I wanted to make a more complicated application which needed to have users, permissions etc. That's how I started learning about backend development. First in Node.js using express but after some time I started using Nest.js as it's very simmilar to angular and I was at the time working as Frontend Angular developer. I also started learning Golang and Rust at some time. In 2022 I got a new job as full stack developer at a startup company. I was writing manily golang but I also did many frontend related work, on one project I did rust and much more. You know how it goes at startups :). I'm very grateful for that, because being thrown into the fire and when there is almost no one to help you is great learning experience.

University

After working for 9 months as frontend developer I've decided to go to university and learn about math, electronics and low-level programming. I was already familiar with C programming, pointers, memory and so on and I loved it, so that's why I wanted to learn about embedded programming, IoT, firmware development and everything else related to that. I'm in a third year now (2024) and so far it was a great experience. At the end of a first year I got a job as full stack developer at local startup company (mentioned above)

Teaching others

Being self thought teached me not to fear new technology and something I don't understand yet and I think that's the greates skill you can have as a programmer. I know I can learn anything and it's an awesome feeling.

Recently I started giving programming lessons and I giving my best to teach people how to think like a programmer. My course is aimed to make someone a programmer in at least 6 months. I cover everything from C, HTML, JS, React, Golang, Git, Docker, Devops topics and really everything that someone needs to be able to an independet programmer. Main idea of the course is to get someone good enough so they can work on some of my projects or their personal ones. I offer everyone an opportunity to work with me on some of my apps. I think that's great for them as they can really use git for example, and see how it feels working in a team of 2-4 people. (It's also great for me :P)

If you have some programming question or want to join my course feel free to contact me.
